# 性能预算

性能预算用于定义“慢到什么程度就必须优化”。

## 当前预算建议

- 应用启动进入可操作状态：10 秒内
- 项目切换：1 秒内
- 代码编辑到预览刷新：200ms 内
- 普通 AI 生成请求：15 到 30 秒
- 超过 30 秒的生成：视为性能退化，需要记录和优化

## 当前已知瓶颈

- 真实模型调用天然比本地模板慢
- 代理链路会增加网络延迟
- 长 HTML 上下文会显著拖慢生成
- 当前缺少流式返回与可取消生成

## 优化优先级

1. 缩短请求上下文
2. 区分快速模式与高质量模式
3. 增加流式返回
4. 增加取消生成
5. 为常见场景缓存或复用局部上下文
