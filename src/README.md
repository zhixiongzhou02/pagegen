# 前端地图

这个目录是 React 应用主体。

## 主要区域

- `components/`：界面组件
- `services/`：Tauri IPC 封装
- `store/`：全局客户端状态
- `lib/`：工具与常量

## 规则

- 大改动前先读本地 README
- service 调用不要散落到小组件里
- 前端工作流改动优先补测试
